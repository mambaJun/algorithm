2. 问题样例

   > 整数 n 的二进制表示中的第 K 位是 几？

   ```java
   public int hammingWeight(int n) {
       int count = 0;
       int k = 31;
       while (k -- >= 0) {
           count += n >>> k & 1;
       }
       return count;
   }
   ```

例子：

```java
public int lengthOfLongestSubstring(String s) {
    int len = s.length();
    if (s == null || len <= 1) return len == 1 ? 1 : 0;

    int res = 0;

    int[] arr = new int[128];
    for (int i = 0, j = 0; i < len; i++) {
        char ch = s.charAt(i);
        arr[ch]++;

        while (arr[ch] > 1) {
            //                如果有相同的，一直向后走
            arr[s.charAt(j++)]--;
        }

        res = Math.max(res, i - j + 1);
    }

    return res;
}
```

总结：

​	双指针 可以 将 O(n^2) 变成 O(n)
